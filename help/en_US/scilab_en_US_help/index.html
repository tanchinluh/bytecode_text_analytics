text_analytics

  • text_analytics
    • getTApath — Gets the full path of the toolbox
    • jar_import — Add jar libraries to the Scilab Java classpath
    • nlp_filterSentiment — Filter a string array based on a given sentiment
    • nlp_getSentiment — Obtains the sentiment of a given string
    • nlp_sentiment — Sentiment analysis of a sentence using Stanford NLP
    • nlp_simple — Runs a specified annotator
    • nlp_tabulSentiment — Tabulate the sentiments
    • nlp_url — Sends an annotation request to a NLP server.
    • pbButton_callback — This function is used by twitter_TweetSentimentDemo.
    • pbSentiment_callback — This function is used by twitter_TweetSentimentDemo.
    • popSentiment_callback — This function is used by twitter_TweetSentimentDemo.
    • text_gencloud — Generate text cloud from a given text-count pairs
    • text_porterstemmer — Apply stemming on the input text
    • text_stripdollar — Strip all dollar sign and replace with the given rep_str.
    • text_stripemail — Strip all emails and replace with the given rep_str.
    • text_striphtml — Strip all HTML and replace with the given rep_str.
    • text_stripnumber — Strip all numbers and replace with the given rep_str.
    • text_stripurl — Strip all urls and replace with the given rep_str.
    • text_tokenize — Tokenize and also get rid of any punctuation
    • twitterStream_init — Creates a Twitter object to connect to Twitter Stream API
    • twitter_TweetSentimentDemo — Tweet sentiment analysis demo
    • twitter_clearStream — Remove tweets retrieved from streaming
    • twitter_displayStream — Display streamed tweets on the Scilab Java Console
    • twitter_getStreamTweets — Import tweets from Twitter stream into Scilab
    • twitter_getTweets — Retrieve tweets from Twitter
    • twitter_init — Creates a Twitter object to connect to Twitter API
    • twitter_processTweets — Perform some text processing on tweets
    • twitter_stopStream — Stops the twitter streaming
    • twitter_stream — Begins the twitter streaming
