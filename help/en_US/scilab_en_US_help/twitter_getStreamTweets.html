twitter_getStreamTweets

Import tweets from Twitter stream into Scilab

Syntax

[id,tweets,info] = twitter_getStreamTweets(twitterstream)
[id,tweets,info] = twitter_getStreamTweets(twitterstream,n)
[id,tweets,info] = twitter_getStreamTweets(twitterstream,n,tweetind)

Parameters

twitterstream :

Java object. Created using twitterStream_init

n :

Double. Number of tweets to be retrived. Default is all are retrieved.

tweetind :

String. Option to retrieve either the latest n tweets ("last") or the earliest n tweets ("first")

id :

Integer64. Tweet ID.

tweets :

List. Tweet text.

info :

Struct. Contains information regarding the tweet.

Description

Retrieve the tweets from the stream stored inside the .

Examples

twitterS = twitterStream_init()
consolebox on
twitter_displayStream(twitterS)
twitter_stream(twitterS,"limit",30)
// After streaming has stopped
[id1,tweets1,info1] = twitter_getStreamTweets(twitterS)
[id2,tweets2,info2] = twitter_getStreamTweets(twitterS,10,"first")
[id3,tweets3,info3] = twitter_getStreamTweets(twitterS,10,"last")
